Diseño de Base de Datos

Fecha de inicio: 15 de Marzo de 2026

Autores: Ivan, Antonella, Lucas,

Descripción del Proyecto

Este proyecto consiste en el diseño y modelado de una base de datos relacional orientada a la administración eficiente de una biblioteca. El objetivo principal es centralizar la información de los ejemplares, autores y categorías para facilitar la búsqueda y consulta de materiales. A través de este sistema, se busca reemplazar los registros manuales por una estructura digital que garantice la integridad de los datos y permita una gestión más ágil del inventario bibliotecario.

Asimismo, el sistema integra un módulo de control para socios y registros de préstamos. El propósito es automatizar la trazabilidad de cada libro, registrando fechas de salida y devolución para evitar pérdidas y optimizar los tiempos de atención. Con esta implementación, se logra un entorno organizado donde tanto bibliotecarios como usuarios pueden conocer en tiempo real la disponibilidad de cada recurso, mejorando la experiencia académica y administrativa.
